A Really Fun HIT

In this experiment you will play a fun communication game with another turker! In each round both of you are going to be given a set of pictures which will look like this:

You'll each have the same 3 or 5 images but the order they are in will be different.

One of you will be assigned the Speaker role and the other will be the Listener . For each round, one of the three objects is the target, which can be identified by a green box around it. Only the speaker can see which object is the target. The task of the speaker is to tell the listener which one of the objects is the target and the listener in turn needs to select the right object based on this information provided by the speaker. Remember that it doesn't make sense to describe the location of the target object, since the order of the images is different for the listener and the speaker.

Only the speaker can see which object is the target for this round.

Thus, the goal is for the listener to identify the correct object based on what the speaker has said. In order to communicate, you're given a cool chatbox where you can send messages back and forth to each other. Here's where the speaker can say which object is the target and the listener can ask for clarifications if what the speaker said is not clear.

Once the speaker sent a message and the listener clicked on the object they believe is the target, the speaker is briefly shown what object the listener clicked on (the clicked object has a red square around it) and you will both be automatically forwarded to the next round of objects. There are a total of 72 rounds. After the 72nd round you will fill out a quick 15 second survey and be on your way.

Two final notes: Since you are playing with another turker, you may see this screen before the game begins:

Just hold tight. Another turker should join the game within the next few minutes.

Finally, please DO NOT refresh the page at any point once you start the game (including the waiting page). If you refresh, you will not be able to rejoin the game you were in and you and your partner may not be able to submit the HIT.
Thank you!
